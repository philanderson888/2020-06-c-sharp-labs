## Build Vue

```js
npm install -g @vue/cli

vue create testapp  // install router and typescript
cd testApp

npm install -g npm  // upgrade
npm install -g yarn 
npm install -g @vue/cli
vue create testapp  // install router and typescript

yarn serve       // or
npm run serve
```
